# 👩‍💻 My Personal Portfolio Website

Welcome to the repository of my personal portfolio website! This project serves as a digital business card, showcasing my journey, skills, and projects as a Quality Assurance (QA) Engineer.

🌐 **Live Demo:** [https://heidipyaricordero.netlify.app/]

---

## 🚀 About the Project

This portfolio is designed to introduce myself to recruiters, tech teams, and the tech community. It highlights my transition into Software Quality Assurance, my technical skillset, and the projects I’ve built along the way. 

As an aspiring QA professional, I treated this portfolio development as a mini-project, ensuring clean code, responsiveness, and a user-friendly layout.

## 🛠️ Tech Stack & Tools

* **Frontend:** HTML5, CSS3, JavaScript
* **Deployment & Hosting:** Netlify
* **Version Control:** Git & GitHub

## 📋 Features

* **Responsive Design:** Optimized for desktop, tablet, and mobile screens.
* **About Me Section:** A brief overview of my passion for tech, QA, and problem-solving.
* **Projects Showcase:** Links and descriptions of my automated and manual testing projects.
* **Contact Form:** A direct way for recruiters and collaborators to reach out.

## 🔍 QA Mindset: How I Tested This Site

Because QA is my passion, this website went through a structured testing process before deployment:
1. **Functional Testing:** Verified that all links, buttons, and the contact form work perfectly.
2. **Cross-Browser Testing:** Tested on Google Chrome, Mozilla Firefox, and Safari to ensure visual consistency.
3. **Mobile Responsiveness:** Checked layouts on various screen resolutions using Chrome DevTools.
